## Setup
- Add `nuxt-multianalytics` dependency using yarn or npm to your project
- Add `nuxt-multianalytics` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    ['nuxt-multianalytics', {
      options: {},
      mixin: function() {}
    }],
 ]
}
```

or

```js
{
  modules: [
    'nuxt-multianalytics',
 ],
 multianalytics: {
   options: {},
   mixin: function() {}
 }
}
```
